    How to see when a user ran a report for the last time in Salesforce

    We can query RecentlyViewed SOQL object in the context of a specific user:

    RecentlyViewed does not have any column which tells about what user it’s about - it always returns data about the user that performs the query. Id contains the id of the recently viewed record (Report in our case), Name contains the record’s name and LastViewedDate contains the date when the record was recently viewed by current user. It turns out that when a user runs a report manually, SFDC marks the Report’s SOQL object as recently viewed. Thanks to that, LastViewedDate from RecentlyViewed object of Type ‘Report’ will contain the date time of when the current user ran the report.
